# Certificates_And_Courses

这个仓库用来保存自己在学习编程的道路上使用过的学习资料.
比如上过的课,参加过的认证课程等等.

争取能保存在这些课程的所有资料.All assignments, demos, and cheat sheets.

当然, 在收集资料时, 为了节约时间, 我肯定会从网络上寻找资料.
这些我肯定会注明出处, 写清楚来源, 以方便自己跟踪后续的资料.
